# 🧪 Tichi Web Application - Manual Testing Project

## 📌 Project Overview

This repository contains the complete Manual Testing documentation for the **Tichi Web Application**. The project was executed using industry-standard QA practices, including test case design, bug reporting, requirement traceability, and test execution reporting.

---

## 🎯 Objectives

- Perform functional testing of the Tichi web application.
- Validate Login and Signup workflows.
- Identify and report application defects.
- Prepare professional QA documentation.
- Demonstrate manual testing skills using real-world scenarios.
